# Phase 1 Secure Messenger

This repository contains the Android client and server for a secure messaging app prototype.

## Build APK using GitHub Actions

1. Push or commit code to the `main` branch.
2. Go to the **Actions** tab in this GitHub repo.
3. Click on the latest workflow run named **Android Build**.
4. Wait for it to finish.
5. Scroll down to **Artifacts** on the right side.
6. Download the `SecureMsg-debug-apk` artifact.
7. Transfer and install the APK on your phone.
